EP01.[工具]Clash的下载、安装和使用

2023年04月12日 技术教程

声明⚠️

本教程旨在于工具分享及使用教程，使用该工具造成的一切不良后果由个人承担

下载地址👾

蓝奏网盘若打不开请进入Github下载

  • Windows & Linux：
    • Github
    • 蓝奏网盘（密码：fc66）
  • Android：
    • Github
    • 蓝奏网盘（密码：fc66）

请自行下载合适的版本

Windows使用教程

  1. 打开安装程序这里根据需求勾选并“下一步”

  2. 填写你要安装的路径并“安装”

  3. 安装完成后运行

界面汉化

汉化补丁：下载链接（密码：fc66）

使用方法：

请自行替换下列路径中的app.asar文件

Clash for Windows\resources\app.asar

开始使用

  1. 确保设置如图中所示

  2. 切换至配置，将文件导入或文件的url填入上方输入框点击下载

  3. 之后切换至相应的配置文件即可使用

推荐的机场节点：

  • IKuuu(免费，每日可领1-2G流量)
  • 七七机场(收费，速度较快，套餐较便宜)、

Android端

与Windows端的操作相似

  1. 打开软件，点击配置

  2. 点击右上角加号，选择相应的选项导入配置文件：本地文件选择“从文件导入”，配置文件链接选择“从URL导入”

  3. 以从URL导入为例，填入URL后点击右上角保存

  4. 返回配置列表，选中导入的配置，回到主页点击启动即可使用
